# The Weatherology App

Forecast Weather APP created for Software Engineering class

## Description

The Weatherology App is a forecast weather application developed as part of a Software Engineering class project. The app is built entirely using Kotlin.

## Technologies Used

- **Kotlin**: 100%

## Features

- Retrieve current weather data for a specified location.
- Display weather forecasts for multiple days.
- User-friendly interface with intuitive navigation.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/favanso/theweatherology_App.git
   ```

2. Navigate to the project directory:
   ```sh
   cd theweatherology_App
   ```

3. Open the project in Android Studio.

4. Build and run the application on an emulator or a physical device.

## Usage

Once the project is set up, you can:

- Enter a location to get the current weather data.
- View the weather forecast for the upcoming days.
